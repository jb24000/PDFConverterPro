📄➡️🖼️ PDF to PNG Converter  
A beautiful, offline-capable desktop web app to convert PDF files into high-quality PNG or JPEG images—right in your browser!
No installation, no server, no privacy worries. Just open the HTML file and go.

✨ Features
Drag & Drop or Click to Upload your PDF
Select Output Quality: Standard, High, Ultra, or Maximum (1x, 2x, 3x, 4x)
Choose Output Format: PNG (lossless) or JPEG (smaller size)
Progress Bar and real-time status updates
Preview each page as an image before downloading
Download individual pages or batch download all as images
Processing statistics: page count, total/average file size, processing time
Clear All button to reset the app
Works 100% offline after first load (no server required)
Modern, responsive UI with smooth animations
Accessibility: Keyboard navigation, ARIA labels, and focus indicators
Dark mode support
🚀 Getting Started
Clone or download this repository:
bash
Copy Code
git clone https://github.com/jb24000/pdf-to-png-converter.git
cd pdf-to-png-converter
Open PDFConverter.html in your browser.
No build step, no dependencies—just open and use!
🖥️ Create a Desktop Shortcut
Windows:

Right-click PDFConverter.html and choose Send to > Desktop (create shortcut).
(Optional) Right-click the shortcut, choose Properties > Change Icon to set a custom icon.
Mac:

Right-click PDFConverter.html and choose Make Alias.
Drag the alias to your Desktop or Dock.
Linux:

Right-click and choose Create Link or use your file manager’s shortcut feature.
🖥️ Usage
Drag and drop a PDF file onto the upload area, or click to browse.
Choose your desired output quality (Standard, High, Ultra, Maximum) and format (PNG/JPEG).
Wait for the conversion—progress is shown.
Preview each page and download as an image, or use the Batch Download button to get all pages at once.
Clear All to reset and start over.
